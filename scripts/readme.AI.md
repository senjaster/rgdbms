Правила написания скриптов
1) Для запуска команд, которые в исходном (old) скрипте выводятся с помощью "set -x" нужно использовать функцию run
2) Команды которые не выводятся (обычно это удаление созданых объектов в начале и конце скрипта) нужно запускать напрямую
3) Если в исходном скрипте нет гиперссылки, не нужно ее "придумывать" - адреса обновились и ты вставляешь неправильные. Если ты считаешь нужным вставить ссылку, то используй link и напиши там, какую ссылку ты хочешь добавить
4) Никогда не пиши "Демонстрация завершена" в конце скрипта
5) Не пытайся читать файлы session.log и timingfile.tm - они бинарные
6) Папка old будет удалена. Копируй из нее нужные файлы на новое место
